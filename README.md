# NLP Programming Assignment 1 (MS)

## Description
This project performs basic text preprocessing and tokenization on
"Alice in Wonderland" text, followed by word frequency analysis.
For master students, a performance comparison between NLTK,
TextBlob, and spaCy is conducted.

## Input
- aliced29.txt

## Output
- cleaned.txt
- words.txt
- top10words.txt
- time_compares.txt

## Libraries
- nltk
- textblob
- spacy
- collections

## How to Run
1. Open Google Colab
2. Run all cells sequentially
3. Output files will be generated automatically
